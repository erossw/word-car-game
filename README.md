# Word Car Game

초등학생을 위한 영어 단어 학습 웹 게임.  
자동차가 자동으로 달리면서 장애물을 피하고, ⭐ 아이템을 먹으면 영어 단어 퀴즈가 나타나 정답을 맞춰야 스테이지를 클리어하는 사이드스크롤 런닝 게임입니다.

## 게임 개요

| 항목 | 내용 |
|------|------|
| 장르 | 사이드스크롤 런닝 + 어휘 퀴즈 |
| 대상 | 초등학교 1~6학년 |
| 플랫폼 | 웹 브라우저 (PC + 모바일) |
| 단어 수 | 801개 (학년별 분류) |
| 비주얼 | 이모지 + CSS |

## 핵심 기능

- 자동 주행하는 자동차 — 점프와 화면 내 앞뒤 이동
- 구덩이, 바위, 선인장 등 장애물 회피
- 단어 아이템 수집 → 영어 퀴즈 팝업
- 4지선다 퀴즈로 정답 선택 → 스테이지 클리어
- **랜덤 단어 시스템**: 같은 스테이지를 재시작해도 항상 다른 단어 출제
- 스테이지 진행마다 속도 증가 및 난이도 상승
- 점수 기록 Top 10 (localStorage 저장)

## 게임 조작

| 입력 | 동작 |
|------|------|
| `Space` / `↑` 방향키 | 점프 |
| `←` / `→` 또는 `A` / `D` | 화면 안에서 앞뒤 이동 |
| 화면 탭 (모바일) | 점프 |
| 모바일 좌우 버튼 | 화면 안에서 앞뒤 이동 |
| 퀴즈 보기 클릭 | 단어 선택 |

## 스테이지 구조

| 스테이지 | 학년 수준 | 단어 예시 | 특징 |
|---------|---------|---------|------|
| 1~5 | 1~2학년 | cat, dog, red, big, rain | 느린 속도, 낮은 장애물 빈도 |
| 6~10 | 3~4학년 | apple, school, doctor, angry | 보통 속도 |
| 11~15 | 5~6학년 | adventure, ecosystem, courage | 빠른 속도 |
| 16+ | 혼합 무한 | 전 학년 랜덤 | 계속 빨라짐 |

## 단어 시스템

801개 단어가 학년별로 분류되어 `src/data/words.js`에 저장됩니다.

| 학년 | 단어 수 | 주요 카테고리 |
|------|--------|------------|
| 1학년 | 140개 | 동물, 색깔, 신체, 사물, 날씨, 기본 동사 |
| 2학년 | 149개 | 음식, 집안용품, 장소, 가족, 계절 |
| 3학년 | 141개 | 학교, 스포츠, 교통, 직업, 감정 |
| 4학년 | 149개 | 자연, 환경, 일상생활, 동사 확장 |
| 5학년 | 96개 | 추상어, 사회, 과학 기초 |
| 6학년 | 126개 | 과학 심화, 사회/문화, 고급 어휘 |

**랜덤 출제 방식**: 게임 시작 시 학년별 단어를 셔플합니다. 스테이지마다 셔플 덱에서 순서대로 꺼내므로 **덱 소진 전까지 중복 없이 출제**됩니다. 전체 소진 시 재셔플하며 직전 마지막 단어와 연속 중복되지 않도록 처리합니다.

## 프로젝트 구조

```
word-car-game/
├── index.html              # 게임 진입점, 퀴즈 오버레이 DOM
├── css/
│   └── style.css           # UI 스타일 (퀴즈 팝업, HUD, 메뉴)
├── src/
│   ├── main.js             # Phaser 3 게임 설정 및 씬 등록
│   ├── scenes/
│   │   ├── BootScene.js    # 씬 초기화 및 에셋 로딩
│   │   ├── MenuScene.js    # 메인 메뉴, 하이스코어 버튼
│   │   ├── GameScene.js    # 핵심 게임 루프
│   │   ├── QuizScene.js    # 퀴즈 DOM 오버레이 제어
│   │   └── ScoreScene.js   # 결과 화면, localStorage 저장
│   ├── objects/
│   │   ├── Car.js          # 플레이어: 점프, 충돌, 무적 처리
│   │   ├── Obstacle.js     # 장애물 스포너 (오브젝트 풀링)
│   │   └── WordItem.js     # 단어 아이템, 반짝임, 퀴즈 트리거
│   └── data/
│       ├── words.js        # 801개 단어 DB + WordManager 클래스
│       └── stages.js       # 스테이지별 속도/빈도/테마 설정
└── docs/
    ├── GDD.md              # 게임 디자인 문서
    ├── architecture.md     # 기술 아키텍처
    └── word-list.md        # 단어 목록 참고 (설계용)
```

## 기술 스택

- **게임 엔진**: Phaser 3 (CDN, 빌드 도구 불필요)
- **렌더링**: HTML5 Canvas + 이모지 Text 오브젝트
- **퀴즈 UI**: HTML DOM + CSS 오버레이 (Phaser 캔버스 위)
- **점수 저장**: localStorage (Top 10, `wordCarGame_scores` 키)
- **단어 관리**: `WordManager` 클래스 (Fisher-Yates 셔플 덱)

## 실행 방법

```bash
# 로컬 서버 실행 (npx 필요)
cd word-car-game
npx serve .

# 브라우저에서 http://localhost:3000 접속
```

또는 `index.html`을 브라우저에서 직접 열어도 됩니다 (단, CORS 제한이 없는 환경).

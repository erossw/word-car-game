## 오답 선택지 풀 (Distractors)

각 단어의 오답 3개는 `words.js`에서 같은 학년 단어 중 무작위로 선택하거나, 아래 카테고리별 오답 풀을 사용합니다.

- 동물 카테고리 단어는 다른 동물 단어로 오답 구성
- 색깔은 다른 색깔 단어로 오답 구성
- 형용사는 반의어를 포함한 형용사로 오답 구성
